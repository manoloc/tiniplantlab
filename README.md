# Tiny Plant Lab website

Static, deployment-ready website for `tinyplantlab.com`.

## Store URLs

- Marketing URL: `https://tinyplantlab.com/`
- Privacy Policy: `https://tinyplantlab.com/privacy/`
- Terms & Conditions: `https://tinyplantlab.com/terms/`
- Support URL: `https://tinyplantlab.com/support/`
- Data deletion URL: `https://tinyplantlab.com/data-deletion/`

## Deploy

Upload the **contents** of this `website` directory to the web root for
`tinyplantlab.com`. The host must serve directory `index.html` files.

The site is plain HTML/CSS and does not require Node.js, a database, or a build
command.

## Before publishing

1. Create or forward `support@tinyplantlab.com`.
2. Create or forward `privacy@tinyplantlab.com`.
3. Replace the “Coming soon” labels with App Store and Google Play links after
   the listings are public.
4. Review the legal pages whenever analytics, advertising, accounts, cloud
   saves, payments, or multiplayer behavior changes.
5. When AdMob is configured, add the official `app-ads.txt` entry supplied by
   Google at `https://tinyplantlab.com/app-ads.txt`.

The legal text is a practical launch draft, not a substitute for advice from a
qualified lawyer familiar with the developer's specific circumstances.
